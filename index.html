📘 Dataset Overview

Loading dataset summary...

This dataset contains detailed records of Women's Tennis Association (WTA) matches played between 2006 and 2025. Each row corresponds to a single match, where Player_1 is always the favorite (lower ranking or odds).

ColumnDescription
TournamentName of the tournament
DateDate of the match
SurfaceType of surface: Hard, Clay, or Grass
RoundStage of the tournament
Player_1 / Player_2Players — Player_1 is the favorite
Rank_1 / Rank_2WTA rankings (lower = stronger)
Pts_1 / Pts_2Ranking points of each player
Odd_1 / Odd_2Bookmaker odds — lower means higher win chance
yTarget variable (1 = favorite won, 0 = lost)
rank_diffRank_2 - Rank_1 (difference in ranks)
pts_diffPts_1 - Pts_2 (difference in ranking points)
odd_diffOdd_2 - Odd_1 (difference in bookmaker odds)
