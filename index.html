Exploratory Data Analysis for WTA matches (2006–2025)

📘 Dataset Overview

This dataset contains detailed information about Women's Tennis Association (WTA) matches between 2006 and 2025. Each row represents a single match.

ColumnDescription
TournamentName of the tournament
DateDate of the match
CourtIndoor or Outdoor court type
SurfaceType of surface: Hard, Clay, or Grass
RoundTournament stage (1st Round, Quarterfinal, etc.)
Player_1 / Player_2Competitors — Player 1 is always the favorite (better rank or lower odds)
Rank_1 / Rank_2Official WTA rankings (lower = stronger)
Pts_1 / Pts_2WTA ranking points of each player
Odd_1 / Odd_2Bookmaker odds — lower means higher chance to win
yTarget variable (1 = favorite won, 0 = lost)
rank_diffDifference in ranks (Rank_2 - Rank_1)
pts_diffDifference in points (Pts_1 - Pts_2)
odd_diffDifference in odds (Odd_2 - Odd_1)
